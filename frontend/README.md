# 🎨 EduMatch 漫改视频生成器前端

## 📁 项目结构

```
frontend/
├── index.html          # 主页面文件
├── App.vue           # Vue 组件（引用在 index.html 中）
├── package.json      # 项目配置
└── README.md        # 说明文件
```

## 🚀 快速开始

### 方法1：直接打开（推荐开发）
1. 直接用浏览器打开 `index.html`
2. 或者使用 Live Server 等工具

### 方法2：使用 serve
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

## 🔧 功能特性

### ✨ 核心功能
- **图片上传**：支持拖拽上传漫画/二次元图片
- **动作描述**：用自然语言描述想要的动画效果
- **风格选择**：日式动漫 / 赛博朋克 / 水墨国风
- **实时状态**：显示任务处理进度
- **视频预览**：生成完成后自动播放
- **视频下载**：一键下载生成的视频

### 🎨 UI 特色
- **渐变背景**：紫色渐变营造科技感
- **毛玻璃效果**：容器半透明模糊背景
- **悬浮动画**：图标和按钮的动效
- **响应式设计**：适配各种屏幕尺寸
- **状态指示**：清晰的进度和状态显示

## 📡 API 接口

### 生成视频
```
POST http://localhost:3000/api/generate
Content-Type: multipart/form-data

Body:
- image: 图片文件
- prompt: 动作描述
- style: 风格选择
```

### 查询状态
```
GET http://localhost:3000/api/status/{taskId}
```

## 🔗 与后端集成

确保后端服务运行在 http://localhost:3000，前端会自动：
1. 上传图片到后端
2. 获取任务 ID
3. 轮询任务状态（每3秒）
4. 显示生成结果

## 🎯 使用流程

1. **上传图片** → 点击上传区域选择图片
2. **描述动作** → 输入想要的动画效果
3. **选择风格** → 选择喜欢的艺术风格
4. **开始生成** → 点击"让图动起来"
5. **等待完成** → 查看实时进度
6. **下载视频** → 保存到本地

## 🎨 样式说明

### 设计理念
- **现代化**：使用 CSS3 特性和渐变效果
- **用户友好**：清晰的交互反馈和状态提示
- **视觉层次**：合理的颜色搭配和空间布局

### 动画效果
- **悬浮效果**：按钮和卡片的上浮动画
- **脉冲动画**：处理状态的闪烁提示
- **浮动动画**：上传图标的上下浮动

## 🛠️ 技术栈

- **Vue 3**：组合式 API 和响应式系统
- **原生 JavaScript**：无构建工具，直接运行
- **CSS3**：现代样式和动画
- **Fetch API**：与后端通信

## 🎉 部署

### 静态部署
1. 将整个 `frontend` 文件夹上传到 Web 服务器
2. 配置服务器支持 `index.html` 作为默认页面
3. 确保跨域访问后端 API

### 注意事项
- 需要配置 CORS 允许前端域名
- 生产环境需要 HTTPS
- 建议使用 CDN 加速静态资源
